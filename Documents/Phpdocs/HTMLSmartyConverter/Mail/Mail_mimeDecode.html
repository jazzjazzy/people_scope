| Usage of this class compared to native php extensions such as | | mailparse or imap, is slow and may be feature deficient. If available| | you are STRONGLY recommended to use the php extensions. | +----------------------------------------------------------------------+

Mime Decoding class

This class will parse a raw mime email and return the structure. Returned structure is similar to that returned by imap_fetchstructure().

USAGE: (assume $input is your raw email)

$decode = new Mail_mimeDecode($input, "\r\n"); $structure = $decode->decode(); print_r($structure);

Or statically:

$params['input'] = $input; $structure = Mail_mimeDecode::decode($params); print_r($structure);
